=======
Amazons
=======

A complete implementation of the Game of The Amazons by Walter Zamkauskas,
including classes for representing the board and game, a graphical interface,
2-person networked play, a formidable AI opponent with multiple difficulty
settings, and an adjustable game clock.

Written by Bishop Wilkins and Justin Gregory for the System Development with
Python course with Joseph Sheedy at EMC Isilon, Q4 2013.
